History
  • History of Java is interesting to know.Members who starts Java Known as Green Team.

  • Java developed by MicroSystems of USA in June 1991.Earlier name of Java is Oak by James Gosling,One of the investor of the language ,other members are Mike Sheridan, Patric Naughton initiated the Java language.
  • It is also known as Sun Engineers Green Team. Java was developed for Development of electronic software devices.
  • Like TVs,VCR etc.

  • The father of the Java language is James Gosling.
  • Language was prepared in 18 months and it was released in 1995 by James Gosling.

  • James Gosling received his Bachelor of Science degree from the University of Calgary
    and completed his M.A. and Ph.D. from Carnegie Mellon University.
