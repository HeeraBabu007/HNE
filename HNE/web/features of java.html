There are many features of Java .Features of Java are easily Understandable.
Features are Given below

Simple

Java is simple because it is Easy to Learn and Write.The mainly use of Java is Bug free Code. Java contain automatically Garbage Collection we do not need to remove unreferenced material. Java have automatically memory management system.If we have knowledge of C++ than we can easily understand
the Java language.Also have a big point which makes Java simple i.e Java does not contain Pointers.

Object Oriented

As we know Java is a object Oriented language means that in Java data is represented by Object.
Object is divided into two parts(methods and fields).Field (Instance variables)represent that what is object.
And methods represent behaviour of object.

In Java we can reuse our code and Java is simple and easily readable language.

Robust

Type checking ,exception handling and garbage collection makes Java robust language.
Java automatically remove the object when it is no longer in use. Does not have problem about memory space.
There have no pointers that makes secure Java.User

Portable

In Java we can run the program on any computer because Java is platform independent.
There are two types of platforms where programs run software and hardware.Java code can be run on multiple
platforms like:Windows, Linux,Mac/OS etc.Without installation of JVM (Java virtual machine) we can not run
the Java program. Java code firstly compiled by the compiler and after that it convert the code in the bytecode.
Write Once and Run Anywhere(WORA).
